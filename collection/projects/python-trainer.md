# Python-тренажёр

**Что это.** Первый кирпич будущей платформы для совместного обучения: сайт на
Django. После Шага 1 это была страница-заглушка; после Шага 2 главная показывает
случайную мотивационную цитату из базы данных. Лежит в виртуальном окружении, под
контролем git, опубликован на GitHub.

**Что уже умеет.** Открывается по адресу `http://127.0.0.1:8000/` и показывает
случайную цитату с автором из базы. Цитатами можно управлять тремя способами: из
кода (view на ORM), из терминала (`shell`; по желанию — своя команда `loadquotes`) и
через админку (мышкой, без кода).

**На чём набил шишки.**
- Порядок важен: сначала `.gitignore`, потом первый коммит — иначе `.venv`
  попадёт в историю.
- venv надо активировать в каждом новом терминале; без приставки `(.venv)`
  команды летят мимо окружения.
- Поменял поля модели — нужны `makemigrations` + `migrate`; правка `__str__`
  миграции не требует.
- Если делал свою команду: она не находилась, пока не положил пустые `__init__.py`
  в `management/` и `commands/`.

**Что дальше.** Автор — пока просто строка (дубли, опечатки). В Шаге 3 вынесем его
в отдельную таблицу и свяжем с цитатами (`ForeignKey`), а дальше — пользователи, их
коллекции и подбор напарников.
